## 使用说明

本页面列出所有支持覆写的字段。

### 修饰符说明

| 修饰符 | 说明 | 适用类型 |
|--------|------|----------|
| `-start` | 添加到列表开头 | List |
| `-end` | 添加到列表末尾 | List |
| `-merge` | 合并键值对 | Map |
| `-force` | 强制替换整个对象 | Object |
| 无 | 简单值直接赋值，对象递归合并 | - |

### 字段书写格式

覆写配置使用 **JSON 格式**，字段名与下方表格中的字段名一致：

```json
{
  "mode": "rule",
  "mixed-port": 7890,
  "dns": {
    "enable": true,
    "nameserver": ["8.8.8.8"]
  },
  "rules-start": ["DOMAIN-SUFFIX,local,DIRECT"],
  "hosts-merge": {
    "test.local": "192.168.1.1"
  }
}
```

## 全局配置

| 字段 | 修饰符 | 说明 |
|------|--------|------|
| `mode` | - | 运行模式（`rule`/`global`/`direct`） |
| `log-level` | - | 日志级别（`silent`/`error`/`warning`/`info`/`debug`） |
| `ipv6` | - | 启用 IPv6 |
| `allow-lan` | - | 允许局域网访问 |
| `bind-address` | - | 绑定地址（`allow-lan: true` 时使用） |
| `external-controller` | - | 外部控制器监听地址 |
| `external-controller-tls` | - | 外部控制器 TLS 监听地址 |
| `external-doh-server` | - | 外部 DoH 服务器地址 |
| `secret` | - | API 访问密钥 |
| `authentication` | `-start` `-end` | HTTP/SOCKS 认证用户列表 |
| `skip-auth-prefixes` | `-start` `-end` | 跳过认证的 IP 前缀 |
| `lan-allowed-ips` | `-start` `-end` | 允许访问的 LAN IP 段 |
| `lan-disallowed-ips` | `-start` `-end` | 禁止访问的 LAN IP 段 |
| `external-controller-cors` | `-force` | 外部控制器 CORS 配置 |
| `geox-url` | - | GeoX 数据库下载地址 |
| `geox-url-force` | `-force` | GeoX 数据库下载地址（强制覆盖） |

## DNS 配置

嵌套对象，直接写字段表示递归合并，使用 `-force` 表示强制覆盖整个对象。

| 字段 | 修饰符 | 说明 |
|------|--------|------|
| `dns.enable` | - | 启用 DNS 功能 |
| `dns.listen` | - | DNS 监听地址 |
| `dns.ipv6` | - | 启用 IPv6 |
| `dns.use-hosts` | - | 使用 Hosts 映射 |
| `dns.use-system-hosts` | - | 使用系统 Hosts |
| `dns.respect-rules` | - | 遵循路由规则 |
| `dns.enhanced-mode` | - | 增强模式（`fake-ip`/`redir-host`/`normal`） |
| `dns.fake-ip-range` | - | Fake-IP 地址段 |
| `dns.fake-ip-range6` | - | Fake-IP IPv6 地址段 |
| `dns.fake-ip-filter-mode` | - | Fake-IP 过滤模式（`blacklist`/`whitelist`/`rule`） |
| `dns.fake-ip-ttl` | - | Fake-IP TTL 值 |
| `dns.ipv6-timeout` | - | IPv6 超时时间（毫秒） |
| `dns.cache-algorithm` | - | 缓存算法 |
| `dns.cache-max-size` | - | 缓存最大大小 |
| `dns.prefer-h3` | - | 优先使用 HTTP/3 |
| `dns.direct-nameserver-follow-policy` | - | 直连 DNS 跟随策略 |
| `dns.nameserver` | `-start` `-end` | 主 DNS 服务器列表 |
| `dns.fallback` | `-start` `-end` | 回退 DNS 服务器列表 |
| `dns.default-nameserver` | `-start` `-end` | 默认 DNS 服务器列表 |
| `dns.fake-ip-filter` | `-start` `-end` | Fake-IP 过滤规则 |
| `dns.proxy-server-nameserver` | `-start` `-end` | 代理服务器 DNS 服务器列表 |
| `dns.direct-nameserver` | `-start` `-end` | 直连 DNS 服务器列表 |
| `dns.nameserver-policy` | `-merge` | DNS 服务器策略映射 |
| `dns.proxy-server-nameserver-policy` | `-merge` | 代理服务器 DNS 策略映射 |
| `dns.fallback-filter` | - | 回退过滤器配置（递归合并） |
| `dns.fallback-filter-force` | `-force` | 回退过滤器配置（强制覆盖） |
| `dns-force` | `-force` | 整个 DNS 配置块强制覆盖 |

### dns.fallback-filter 子字段

| 字段 | 修饰符 | 说明 |
|------|--------|------|
| `dns.fallback-filter.geoip` | - | 启用 GeoIP 判断 |
| `dns.fallback-filter.geoip-code` | - | GeoIP 国家代码 |
| `dns.fallback-filter.ipcidr` | `-start` `-end` | IP 段列表 |
| `dns.fallback-filter.geosite` | `-start` `-end` | GeoSite 域名列表 |
| `dns.fallback-filter.domain` | `-start` `-end` | 域名列表 |
