## 特性

- Mihomo 内核
- Sub-Store 可选支持
- Web 面板
- 更多请自行探索...

## 适配性

- Android 7.0 及以上
- 支持 `armeabi-v7a`、`arm64-v8a`、`x86`、`x86_64` 架构
